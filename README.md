## Название плагина
time/Moscow

## Описание
Плагин для Visual Studio Code, разработанный в рамках лабораторной работы №3.
Плагин отображает текущее московское время при вызове пользовательской команды.

## Возможности
- Регистрация пользовательской команды в VS Code
- Получение текущего времени в часовом поясе Moscow
- Отображение времени во всплывающем уведомлении

## Используемые технологии
- Visual Studio Code
- Node.js
- TypeScript
- VS Code Extension API

## Запуск
Плагин запускается в режиме отладки расширений Visual Studio Code (Run Extension).
Команда вызывается через Command Palette (ctrl + shift + p).
